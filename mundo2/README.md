# Curso Python 3 – Mundo 2 (Curso em Vídeo)

Este repositório contém os exercícios e práticas realizados durante o **Curso de Python 3 – Mundo 2**, ministrado pelo professor **Gustavo Guanabara** no Curso em Vídeo.

## Conteúdos estudados

* Estruturas condicionais compostas
* Estruturas de repetição (`for`)
* Estruturas de repetição (`while`)
* Controle de fluxo (`break` e `continue`)
* Contadores e acumuladores
* Validação de dados
* Lógica de repetição em algoritmos

## Objetivo

Consolidar o uso de **estruturas de controle** em Python, desenvolvendo a capacidade de resolver problemas utilizando loops e condições.

Curso finalizado com foco em fortalecer a **lógica de programação e o raciocínio algorítmico**.
